# PRAQEN - P2P Bitcoin Trading Platform

**Peer-to-peer crypto payments give you access to bank accounts and mobile wallets worldwide**

## 🌟 Features

- **Buy & Sell Bitcoin** - P2P marketplace with real-time rates
- **Escrow Protection** - Bitcoin locked until payment is confirmed
- **Multiple Payment Methods** - MTN Mobile Money, Vodafone Cash, Bank Transfer, and more
- **Dispute Resolution** - Moderator system for fair trade resolution
- **User Ratings & Reviews** - Build trust through trading history
- **Real-time Chat** - Communicate directly with trading partners
- **Mobile Responsive** - Works on all devices

## 🚀 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Lucide Icons
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- Supabase (PostgreSQL)
- JWT Authentication
- Nodemailer (Email verification)
